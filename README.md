# botbotbot

Проект "ботботбот" - это бот для учеников, в нем собранно расписание для обучающихся с 5 по 11 классы, которое поможет школьников знать уроки. Бот имеет как команды, так и кнопки, интерфейс удобный и в нем лекго разобраться, ничего лишнего. 

При нажатии кнопки "My schedule" - вам предаставят кнокпки классов, вам нужно будет выбрать один из них и бот покажет вам расписание 

При нажатии кнопки "All schedule" - бот вам педоставит расписание всех классов, с 5 по 11
